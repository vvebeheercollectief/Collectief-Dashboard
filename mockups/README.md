# Mockups & schetsen (archief — niet live)

Losse HTML-schetsen uit het ontwerpproces van het Collectief Dashboard en de website. **Geen onderdeel van de live-app** (de app = `index.html` + `src/` + `styles.css` in de hoofdmap).

- `mockup-*.html` — ontwerp-iteraties van dashboard-onderdelen (Vandaag, bulk-acties, logboek, VvE-pagina, AI-hulp, commandocentrum, UX-snelheid).
- `mockup-design-*.html` — visuele richtingen voor het "Operator"-herontwerp (live sinds 2026-06-17).
- `website-mockup-*.html` — ontwerp-iteraties voor de marketingwebsite (zie aparte `vve-website`-repo).
- `presentatie-*.html` — eenmalige presentaties (code-review, routekaart).
- `brand-preview.html`, `brand-update-preview.html` — merk/stijl-previews.

Verplaatst hierheen op 2026-06-19 om de hoofdmap op te schonen (Batch D van het werkwijzen-verbeterprogramma). Veilig te verwijderen als je ze niet meer nodig hebt. De wekelijkse weekpresentatie-tool `vrijdagsoverzicht.html` staat bewust nog in de hoofdmap.
