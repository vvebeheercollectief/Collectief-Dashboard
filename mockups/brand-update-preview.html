Overzicht van alle aanpassingen

1. Pagina-achtergrond
Van blauw-grijs naar neutraal warm grijs, passend bij het navy merk.
#f0f4f8 → #F0F2F5
2. Sidebar gradient
Subtiel verticaal gradient geeft diepte aan de sidebar.
flat #2B3544 → gradient #2B3544 → #243040
3. Logo-icoon gradient
Logo krijgt een fris teal-gradient i.p.v. navy→teal.
gradient navy→teal → gradient #0D7377 → #14B8A6
4. Actief sidebar-item
Actief item krijgt een linkerbalkje en rijkere gradient achtergrond.
rgba(13,115,119,.25) → gradient + border-left #14B8A6
5. Oppakken (blauw)
Dieper, warmer blauw dat harmonieus bij navy past.
#2563eb / #dbeafe / #bfdbfe → #1D6AA5 / #E0EEF9 / #B8D4EC
6. Vergaderverzoeken (amber)
Dieper amber, minder schreeuwerig.
#d97706 → #B45309 (lichte tinten blijven)
7. Offerte-trajecten (paars)
Warmer paars dat beter past bij het teal accent.
#7c3aed / #ede9fe / #ddd6fe → #6D5BD0 / #EEEAFD / #D4CCFA
8. LOD (rood)
Dieper rood, professionelere uitstraling.
#dc2626 → #B91C1C
9. Groen (afgerond)
Iets dieper zodat het niet met teal accent concurreert.
#059669 → #047857
10. Pink (Ontwikkeling/Meldingen)
Dieper roze, rijkere toon.
#db2777 → #BE185D
11. Badge gradient
Sidebar badges krijgen een teal-gradient voor meer premium gevoel.
flat #0D7377 → gradient #0D7377 → #14B8A6
12. Stat card shadows
Gekleurde subtiele schaduw op stat cards, matcht de sectie-kleur.
rgba(0,0,0,.08) → rgba(kleur,.08)
